* 誰にでも合うようには作っていないので、各自で修正して下さい。

  * arydshln.sty (https://www.biwako.shiga-u.ac.jp/sensei/kumazawa/tex/arydshln.html)
  * emathT.sty (http://emath.s40.xrea.com/teisei.htm 訂正版zip )
* 自己責任でお使い下さい。このフォーマットを使って不利益を被ったとしても責任を負いかねます。
